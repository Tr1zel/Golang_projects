**Система управления задачами (To-Do CLI)**

- **Пример:** `todo add "Купить молоко"` → сохранение в JSON
#### **Цель:**

Создать CLI-приложение для управления списком задач.

#### **Функциональные требования:**

- `add "Название задачи"` — добавить задачу.
- `list` — вывести список задач.
- `done 1` — отметить задачу как выполненную.
- `delete 1` — удалить задачу.
- Хранение задач в JSON-файле (например, `todo.json`).

#### **Технологии:**

- `time` — для работы с дедлайнами задач.
- `os` — для работы с файлами.
- `encoding/json` — для сохранения/чтения задач.

#### **Пример использования:**

```
todo add "Купить молоко"
todo list
todo done 1
```
